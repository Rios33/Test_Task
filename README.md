Test_Task

Утилита проверяет доступность указанных хостов и измеряет время ответа.
Можно передавать один адрес или несколько через параметр, либо загрузить список из файла.
Результат выводится в консоль. При необходимости данные можно сохранить в файл.

Запуск

Запуск выполняется через Run/Debug Configurations в PyCharm.
В поле параметров указываются аргументы, например:

-H https://ya.ru,https://google.com -C 5

После сохранения конфигурации утилита запускается обычной кнопкой Run.

Параметры

-H
Список адресов через запятую.

-F
Путь к файлу со списком хостов.

-C
Количество запросов к каждому адресу.

-O
Сохранение результата в указанный файл.

Пример результата

Host: https://ya.ru
Success: 5
Failed: 0
Errors: 0
Min: 1.8639 s
Max: 2.4236 s
Avg: 2.0593 s
